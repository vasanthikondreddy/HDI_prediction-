# User Manual

## Steps to Use

1. Open the Flask application in your browser.
2. Enter the required HDI indicators:
   - Life Expectancy
   - Mean Years of Schooling
   - Expected Years of Schooling
   - GNI per Capita
3. Click the **Predict** button.
4. View the predicted HDI category.

## Features

- Simple interface
- Fast prediction
- Easy navigation
- Responsive design
