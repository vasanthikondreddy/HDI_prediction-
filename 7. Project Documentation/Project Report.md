# Project Report

## Title

A Comprehensive Measure of Well-Being: Human Development Index (HDI) Prediction using Machine Learning

## Objective

Develop a Machine Learning model capable of predicting HDI categories based on socio-economic indicators and deploy it using Flask.

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript

## Outcome

The project successfully predicts HDI categories through a web application, demonstrating the practical application of Machine Learning in socio-economic analysis.
