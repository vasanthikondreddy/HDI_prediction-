# User Interface Design

## Home Page

The application provides a clean and responsive interface where users can enter:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- Gross National Income (GNI)

## Prediction

After clicking the Predict button, the application displays the predicted Human Development Index (HDI) category.

## Design Goals

- Simple layout
- Easy navigation
- Responsive design
- Fast prediction
- Clear output presentation
