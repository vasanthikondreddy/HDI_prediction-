# Dataset Description

## Overview

The Human Development Index (HDI) dataset contains socio-economic indicators used to classify countries into different human development categories.

## Dataset Features

- Life Expectancy
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income (GNI) Per Capita

## Target Variable

HDI Category

- Low
- Medium
- High
- Very High

## Dataset Usage

The dataset is used to train a machine learning model that predicts the HDI category based on the input indicators.

## Source

United Nations Development Programme (UNDP) Human Development Reports.
