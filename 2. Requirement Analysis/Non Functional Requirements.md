# Non Functional Requirements

## Performance
- Prediction should complete within a few seconds.

## Reliability
- The application should provide consistent prediction results.

## Usability
- The interface should be simple and easy to navigate.

## Maintainability
- The project should be modular and easy to update.

## Scalability
- Future Machine Learning models can be integrated without major code changes.

## Portability
- The project should run on Windows, Linux, and macOS.

## Security
- Input validation should prevent invalid data from affecting predictions.
