# Functional Requirements

## Overview

The HDI Prediction System enables users to predict the Human Development Index (HDI) category based on socio-economic indicators.

## Functional Requirements

### FR-1 User Input
The system shall allow users to enter:
- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- Gross National Income (GNI) per Capita

### FR-2 Data Validation
The system shall validate user inputs before prediction.

### FR-3 Prediction
The system shall predict the HDI category using the trained Machine Learning model.

### FR-4 Display Result
The system shall display the predicted HDI category on the web interface.

### FR-5 User Interface
The system shall provide a simple, responsive, and easy-to-use interface.

### FR-6 Model Loading
The application shall load the trained model automatically during startup.
