# Test Plan

## Objective

The objective of testing is to verify that the HDI Prediction application functions correctly and provides accurate predictions based on user inputs.

## Scope

The following components were tested:

- User Interface
- Input Validation
- Machine Learning Model
- Flask Backend
- Prediction Output

## Testing Types

- Functional Testing
- Integration Testing
- User Interface Testing
- Model Prediction Testing

## Expected Outcome

The application should accept valid inputs, predict the correct HDI category, and display the results without errors.
