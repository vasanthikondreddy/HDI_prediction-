# Use Cases

## Government Agencies

Predict the development category of a country based on health, education, and income indicators.

## Researchers

Analyze development trends and compare countries using predicted HDI categories.

## Students

Understand how Machine Learning can be applied to socio-economic data.

## Policy Makers

Evaluate potential improvements in development indicators before implementing policies.

## Educational Institutions

Use the project as a practical demonstration of Machine Learning applications.
