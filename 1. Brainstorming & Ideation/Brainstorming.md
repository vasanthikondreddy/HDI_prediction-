# Problem Statement

## Title
A Comprehensive Measure of Well-Being: Human Development Index (HDI) Prediction using Machine Learning

## Problem Description

The Human Development Index (HDI) is one of the most widely used indicators to measure the overall development of a country. It combines three important dimensions:

- Life Expectancy
- Education
- Gross National Income (GNI) per Capita

Traditional evaluation of HDI requires extensive statistical analysis and historical data processing. Governments, researchers, and policy makers need faster methods to estimate the development category of a country based on available indicators.

This project proposes a Machine Learning based web application that predicts the HDI category using socio-economic indicators. The system simplifies the prediction process and provides quick, reliable results through an interactive Flask web application.

## Proposed Solution

Develop a Machine Learning model capable of predicting the HDI category from key development indicators and deploy it using Flask with a user-friendly interface.
