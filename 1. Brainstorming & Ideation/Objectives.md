
# Objectives

## Primary Objective

To develop a Machine Learning based web application capable of predicting the Human Development Index (HDI) category using socio-economic indicators.

## Specific Objectives

- Collect and preprocess HDI data.
- Perform exploratory data analysis.
- Train multiple Machine Learning models.
- Compare model performance.
- Select the best-performing model.
- Save the trained model.
- Develop a Flask web application.
- Design an intuitive user interface.
- Enable users to predict HDI categories instantly.
- Provide an easy-to-use system for educational and research purposes.
